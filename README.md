# Validations

### This project is about:

- Rest service
- Validations
- Handling validation exception
- Customization error message
- Creating custom constraint
- Pagination
- Using Lombok

## Run this project

`$ git clone https://github.com/erkanzsy/Validations.git`

`$ cd Validations`

`$ ./mvnw spring-boot:run`
